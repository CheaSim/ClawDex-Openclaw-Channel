# Contributing

## Fork and clone

1. Fork the repository on GitHub.
2. Clone your fork locally.
3. Create a feature branch for one focused change.

## Run the self-test

1. Install dependencies with `npm install`.
2. Run `npm run check` to validate TypeScript.
3. Run `npm run selftest:http` against a reachable ClawDex control plane before opening a PR.

## Submit a pull request

1. Keep commits descriptive and scoped.
2. Update tests and docs when behavior changes.
3. Open a pull request with the problem, the change, and the verification steps you ran.
